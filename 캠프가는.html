1590번 캠프가는 영식

#수학 #브루트포스 알고리즘 #이분 탐색


  • Python

  • Java

  • JavaScript

  • Dart
출처: https://www.acmicpc.net/problem/1590
어떤 버스의 시작 시각이 특점 시점을 기준으로 10분 후이고, 간격은 10분이고, 대수가 5대이면,
이 버스는 10분, 20분, 30분, 40분, 50분에 한 대씩 출발한다.

영식이가 도착했을 때,
첫 차가 출발하지 않았으면 첫 차를 타고
출발하지 않은 시간대의 차가 있으면 가장 빨리 오는 차를 타고
마지막 버스까지 놓쳤다면 차를 탈 수 없다. ('-1' 출력)
