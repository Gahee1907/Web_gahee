동전게임

10837번 동전게임

#수학 #애드혹


  • Python

  • Java

  • JavaScript

  • Dart
출처: https://www.acmicpc.net/problem/10837

규칙은 다음과 같다.
1. 한 라운드에서 영희와 동수는 한 번씩 동전을 던지고 항상 영희가 먼저 던진다.
2. 동전을 던져 앞면이 나오면 1점을 얻고, 뒷면이 나오면 점수를 얻지 못한다.
3. 한 명이 남은 기회에 모든 점수를 얻더라도 상대방이 현재까지 얻은 점수보다 작게 되면 게임 도중 어떤 시점에서도 게임은 바로 끝난다.

영희와 동수의 점수가 될 수 없는 두가지 경우가 존재한다. (이 경우에 '0' 출력)
영희의 점수가 동수의 점수보다 작을 때, (라운드 - 영희점수 + 동수점수 + 1)이 동수의 점수보다 작을 경우
영희의 점수가 동수의 점수보다 클 때, (라운드 - 영희점수 + 동수점수 +2)가 영희의 점수보다 작을 경우
위 상황의 반대의 경우와 영희와 동수가 동점일 경우에는 '1'을 출력한다.
