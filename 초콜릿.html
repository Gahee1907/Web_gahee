초콜릿 자르기

2163번 초콜릿 자르기

#수학 #사칙연산


  • Python

  • Java

  • JavaScript

  • Dart
출처: https://www.acmicpc.net/problem/2163

1x1 크기의 초콜릿으로 쪼개기 위한 최소 쪼개기 횟수를 구해야 한다.
세로방향으로 자른다면 m값-1 번이 진행된다. 가로도 방향도 마찬가지로 n값-1 번 진행된다.
여기에 전체적으로 m개만큼의 조각이 있기 때문에 m*(n-1)만큼 진행된다.

그래서 최종적으로 자르기 횟수는 m*n-1번이 된다.
